#AI Email Assistant
An intelligent email management system with AI-powered automation.

## Features (In Development)
- Gmail Integration
- AI email summarization
- Smart task extraction
- telegram notifications
- Calendar integration

## Tech Stack
- Backend: FastAPI
- Database: PostgreSQL
- Cache: Redis
- AI: Hugging Face / Groq
- Frontend: Streamlit
- Deployment: Docker
